An Q Tran

The concept of this game is based on the World's Hardest Game. Player plays as the green box and the goal is to move into the cyan area. When ever the player collides with the enemies, the player returns to the beginning and death count increases. 

Player can use 4 buttons to move the box: UP, DOWN, LEFT, RIGHT. And player can go back to the title by press SELECT key.
